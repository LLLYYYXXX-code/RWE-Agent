# MIMIC Research Report Writer

## Core Responsibilities

Based on the outputs from `stats_analysis` and `figure_report`, automatically generate a clinical research report that conforms to academic standards.

## Trigger Conditions

- User requests generation of a final research report
- Executed as the final step in the `rwd_to_rwe_statistical_workflow` pipeline

## Input Requirements

1. **Statistical Analysis Results**: Statistical summaries generated by `stats_analysis` (e.g., effect sizes, confidence intervals, p-values, etc.)
2. **Figure Files**: Chart paths or descriptions generated by `figure_report`
3. **Study Design Information**: PICO elements, cohort definitions, covariates, etc., from `task_contract` or `cohort_builder`

## Report Structure

### 1. Title Page
- Study title (auto-generated based on PICO)
- Author/institution (default: "MIMIC-IV Research Group")
- Date

### 2. Abstract
- **Background**: Brief description of the clinical problem and research necessity
- **Methods**: Data source, study design, cohort definition, primary endpoint, statistical methods
- **Results**: Key figures (e.g., cohort size, event counts, effect sizes with confidence intervals)
- **Conclusion**: One-sentence summary of clinical significance

### 3. Introduction
- Disease background and clinical controversy
- Existing evidence and knowledge gaps
- Research objectives and hypotheses

### 4. Methods
- **Data Source**: MIMIC-IV v2.2
- **Study Design and Population**: Inclusion/exclusion criteria, exposure and comparator definitions
- **Outcome Measures**: Primary and secondary endpoint definitions
- **Covariates**: List of variables used for adjustment or matching
- **Statistical Methods**: Specific methods such as propensity score matching, Cox regression, IPTW, etc., and software used (R/Stata)

### 5. Results
- **Baseline Characteristics**: Tabular description of population characteristics before and after matching
- **Primary Analysis**: Effect sizes, 95% CI, p-values, with references to figure/table numbers
- **Secondary Analyses**: Subgroup or sensitivity analysis results
- **Figure References**: e.g., "Figure 1 shows the Kaplan-Meier survival curve"

### 6. Discussion
- **Summary of Key Findings**
- **Comparison with Previous Studies**
- **Clinical Implications and Potential Mechanisms**
- **Limitations** (e.g., residual confounding, single-center, coding accuracy, etc.)
- **Future Directions**

### 7. Conclusion
- Concise restatement of the main conclusion

### 8. References
- Auto-generated placeholders, with prompts for the user to supplement

## Output Format
- Markdown format, suitable for subsequent conversion to PDF or Word
- Figure references use placeholders, e.g., `![Kaplan-Meier Curve](path/to/figure.png)`

## Important Notes
- All statistical results must be strictly based on input data; fabrication is prohibited
- Language style should be objective and academic
- If input data is incomplete, clearly mark missing sections in the report
